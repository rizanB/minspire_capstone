# minspire_capstone
This repo holds the source code and materials for the MINspire capstone 2026.
Nepal's trekking industry draws hundreds of thousands of visitors each year, yet trekkers lack a structured, data-driven way to discover routes that match their individual needs. Existing recommendations are largely informal, making it difficult to account for preferences such as difficulty, duration, altitude, cost, and season simultaneously.
This project addresses three core challenges using the Nepal Trekking Dataset:
(1) Trek recommendation: suggesting the most suitable trek based on user preferences using cosine similarity and KNN-based persona matching
(2) Cost and Duration prediction: modeling trek characteristics using linear regression with regularization and decision tree regressors.
(3) Hidden gem detection — identifying underrated, high-value treks that are statistically anomalous using isolation forest.
Together, these approaches aim to make Nepal's trekking landscape more accessible and navigable for both novice and experienced trekkers.
